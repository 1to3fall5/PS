# PS
PS save script
Direct save file as TGA

1. DSAT 会保存为TGA，文件名和保存位置与psd文件一致，不会跳任何弹窗
2. DSAT2 会保存为TGA， 会跳一个弹窗选择位置和名称
3. DSAT3 会保存为32位的TGA， 会跳一个弹窗选择位置和名称

把脚本放入PS的脚本目录，例如C:\Program Files\Adobe\Adobe Photoshop 2022\Presets\Scripts
然后设置好相应的快捷键使用即可
