# PS
PS save script
Direct save file as TGA

1. DSAT 会保存为TGA，文件名和psd一致， 保存位置和psd文件位置一致
2. DSAT1 会保存为TGA， 会跳一个弹窗选择位置和名称
3. DSAT2 会保存为32位的TGA， 会跳一个弹窗选择位置和名称

把脚本放入PS的脚本目录，例如C:\Program Files\Adobe\Adobe Photoshop 2022\Presets\Scripts
然后在快捷键设置界面
